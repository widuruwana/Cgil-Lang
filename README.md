# Cgil

A high-fantasy RPG-themed systems programming language that transpiles to C.
Designed for bare-metal x86 OS development.

**Status:** Lexer complete. Parser in progress.

## Pipeline
Lexer → Parser → AST → C CodeGen

## File Extension
`.gil`
